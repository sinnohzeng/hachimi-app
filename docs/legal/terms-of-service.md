## 2. Description of Service

Hachimi is a habit-tracking application that combines focus timer functionality with a virtual cat-parenting game. Core features include:

- **Habit management**: Create and track daily habits with focus goals
- **Focus timer**: Countdown and stopwatch modes for focused work sessions
- **Cat-parenting game**: Adopt virtual cats that grow as you build habits
- **Achievement system**: Earn achievements and virtual coins through consistent practice
- **AI features** (optional): Chat with your virtual cat and generate diary entries powered by cloud AI
