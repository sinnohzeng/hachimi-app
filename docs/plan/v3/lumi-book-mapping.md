## 四、灵感清单完整内容索引

### 4.1 习惯养成范例（9 个，均含四法则设计）

每个范例包含：看得见、想去做、易上手、有奖励 + 行动宣言模板

1. **提前 30 分钟睡觉** — 手机闹钟 / 听轻音乐 / 先尝试提前 15 分钟 / 第二天醒来状态更好
2. **每天喝 __ 升水** — 床头放杯水 / 用喜欢的水杯 / 先喝一杯也可以 / 身体更有能量
3. **阅读 __ 页书** — 把书放在会经过的位置 / 选真正喜欢的书 / 只读五页也行 / 获取知识+放松
4. **感恩日记** — 把日记本放床头 / 可以装饰页面 / 太累时写一件事也可以 / 翻阅记录感受进步
5. **姿势提醒** — 便利贴"坐直了！" / 与朋友互相监督 / 看到就调整两秒 / 身体更舒服气质提升
6. **扔垃圾** — 晚饭后设闹钟 / 在路上播放喜欢的歌 / 只扔一袋也可以 / 干净空间+小奖励
7. **整理房间** — 手机闹钟提醒 / 播放喜欢的音乐 / 只整理一个小区域 / 轻松清爽的环境
8. **无电子设备时间** — 设"无电子设备"闹钟 / 准备离线活动 / 先尝试两分钟不碰手机 / 睡得更好专注力更强
9. **洗衣服** — 手机闹钟提醒 / 选喜欢的洗涤剂听音乐 / 只洗几件也可以 / 清爽衣服带来肯定

### 4.2 爱自己活动（~50 条）

分类来自原书云朵气泡布局：

**自我关怀类**：尝试做新的菜肴、买喜欢的文具、写下自己的优点、对自己说"我爱你"每天至少一次、为自己泡一杯暖暖的茶、清理房间断舍离、参加一次读书会、参观一次艺术展览

**生活享受类**：买一份想吃的零食、拍一张大笑的照片、去一直想去的餐厅享受一顿美食、给自己做一次丰盛的早餐、买一些漂亮的明信片给自己寄一张、买一束喜欢的花、学习一项新的运动、去海边发呆

**购物犒赏类**：买一套舒适的睡衣、给自己放一天假、写成功日记、去郊外散步呼吸新鲜空气、泡个热水澡加些香薰、整理照片制作成相册、参加一次户外野餐、种一盆植物用心照料它

**美食创意类**：做一顿丰盛的晚餐、买一双舒适的鞋子、学习摄影记录生活中的美好瞬间、去唱卡拉 OK 尽情释放自己、去动物园或植物园游玩、看一部温馨的电视剧或电影、参加一次志愿者活动、学习绘画

**探索挑战类**：去参加一次音乐节、尝试拒绝一次、看一场体育比赛、学习剪辑视频、参加一次徒步旅行、尝试射箭、尝试冥想放松身心

### 4.3 年度成长计划灵感（8 维度）

| 维度 | 图标 | 灵感举例 |
|------|------|---------|

## 五、电子化决策记录

### 5.1 不电子化的内容

| 原书内容 | 不电子化原因 | 替代方案 |
|---------|-----------|---------|
| 点阵涂鸦区 | 手机上自由绘画体验差 | 改为自由文本输入区 |
| 年度谢谢卡 | 实体卡片的触感价值无法数字还原 | 融入年度寄语"给亲爱的未来的我" |
| 每日看板时间轴（6:00-24:00 详细格）| 在手机上逐格填写体验差 | 简化为"今日一点光"一句话记录 |
| 猫咪月封面手绘（每月不同）| 需要 6 套原创插画 | Phase 4 考虑，MVP 用简化版 |

### 5.2 电子化升维的内容

| 原书内容 | 纸质局限 | 电子升维 |
|---------|---------|---------|
| 心情追踪圆圈 | 手动画表情，无法统计 | 自动统计 5 种心情分布 + 热力图 |
| 烦恼处理器 | 写满了无法扩展 | 无限条目 + 自动归档 |
| 100 天挑战 | 手动划格子 | 自动打卡 + 里程碑动画 + 奖励 |
| 灵感清单 | 附赠小册子，翻阅不便 | 内嵌提示词，空页面自动显示 |
| 年历 | 翻到前头才能看 | 任何时候一键切换到年度视图 |
